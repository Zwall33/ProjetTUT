Bienvenue sur le site dédié au jardin connecté

Notre site est un projet tutoré réalisé à la Licence professionnelle SARII de Bordeaux.
Ce projet a pour but de contrôler un jardin connecté. Les capteurs répartis dans les quatres bacs du jardin collecte des informations sur :

  • la température
  • l'humidité
  • l'intancité lumineuse
  • la fertilité du sol
Le calendrier indique les cycles des récoltes du jardin et permet l'envoie de notifications(optionnelle).
Ce site permet aussi de contrôler des pompes, on peut ainsi :
  • programmer les pompes grâce au calendrier
L'ensemble de ces données sont presente dans notre base de donnée.
